# RecursiveTextSplit

Swift port of LangChain's [`RecursiveCharacterTextSplitter`](https://github.com/langchain-ai/langchain/blob/master/libs/text-splitters/langchain_text_splitters/character.py).

Designed for extraction into a standalone package. No Foundation Models dependency — only `Foundation`.

## Usage

```swift
import RecursiveTextSplit

let splitter = RecursiveCharacterTextSplitter(
    config: .langchainDefault(chunkSize: 500, chunkOverlap: 50)
)
let chunks = splitter.split(text: longDocument)
```

## Conformance tests

Golden fixtures in `Tests/RecursiveTextSplitTests/Fixtures/langchain_golden.json` are generated by:

```bash
python3 Scripts/generate-splitter-fixtures.py
```

CI runs Swift tests against committed fixtures only (no Python required on the runner).
